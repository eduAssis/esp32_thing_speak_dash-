Monitoramento de Temperatura e Umidade

Objetivos:

  • Coletar dados de temperatura e umidade em tempo real utilizando o microcontrolador ESP32 e o sensor DHT11.
  • Transmitir esses dados para a plataforma ThingSpeak para visualização.
  • Acionar um relé caso a temperatura ultrapasse 31 °C ou a umidade relativa do ar seja superior a 70%.

Explicação sobre o funcionamento do projeto, incluindo a configuração do ESP32, o sensor DHT11, e o envio dos dados para a plataforma ThingSpeak.
