Monitoramento de Temperatura e Umidade

Esta atividade avaliativa envolve a coleta de dados de temperatura e umidade usando um sensor DHT11, que é transmitido para a plataforma ThingSpeak. O sistema também controla um relé com base em condições de temperatura e umidade.

Objetivos:

  • Coletar dados de temperatura e umidade em tempo real.
  • Transmitir esses dados para a plataforma ThingSpeak para visualização.
  • Acionar um relé caso a temperatura ultrapasse 31 °C ou a umidade relativa do ar seja superior a 70%.

Funcionamento

O sensor DHT11 mede a temperatura e a umidade do ambiente. Os dados são enviados para o ThingSpeak, onde podem ser visualizados em tempo real. O relé é ativado quando:

  • A temperatura excede 31 °C.
  • A umidade relativa do ar ultrapassa 70%.
